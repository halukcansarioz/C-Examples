## Installation & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/halukcansarioz/C-Examples.git
```

### 2. Navigate to the Project Directory
```bash
cd C-Examples
```

### 3. Compile and Run Any Code
Each file is a standalone example. For example:

**Linux / macOS:**
```bash
gcc "Faktoriyel hesaplama.c" -o faktoriyel
./faktoriyel
```

**Windows (MinGW / GCC must be installed):**
```bash
gcc "Faktoriyel hesaplama.c" -o faktoriyel.exe
faktoriyel.exe
```

> **Note:** File names contain Turkish characters and spaces, so it is recommended to use quotes (`""`) when running them in the command line. Alternatively, you can open and run the files directly using an IDE (VS Code, Code::Blocks, Dev-C++).
